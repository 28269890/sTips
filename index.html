插件简单说明

sTips是针对标签属性 alt 和 title 的增强插件。
title属性优先于alt属性，被本插件激活功能的元素的这两项属性将被删除。
提示内容框结构:
sTipsTop
sTipsBody
默认情况下sTipsTop和sTipsBody的样式是空的，只有最外层的div做了简单的样式。
sTipsTop和sTipsBody的内容由 alt 或 title 的内容中获取。
alt 或 title的值以{||}分割为sTipsTop和sTipsBody的值。
sTipsBody以ajax:开头则会提示ajax:后面的地址返回的html数据。
sTipsBody以fn:开头则会返回fn:后的函数的返回值。
固定提示位置：(position的值)
1：左上角对齐 11：上方显示对齐左边框 21：上方显示对齐右边框 2：右上角对齐
12：左侧显示对齐上边框 提示元素 22：右边显示对齐上边框
41：左侧显示对齐下边框 31：右侧显示对齐下边框
4：左下角对齐 42：下方显示对齐左边框 32：下方显示对齐右边框 3：右下角对齐
跟随鼠标位置：(mouse的值)
1 2
鼠标位置
3 4

默认值

JS Code: 
	
{
	phy:1, //透明度 0-1
	inTime:200,//淡入时间
	outTime:200,//淡出时间
	sTipsCss:"sTips",//给外框指定css样式
	topCss:"sTip-Top",//给提示框头部指定css样式，值为css名称
	bodyCss:"sTips-Body",//给主提示框指定css样式，值为css名称
	ajaxError:"",//ajax错误提示开关，唯一值：off
	ajaxLoading:"loading...",//ajax读取之前的信息，也可修改成图片例如：< img src="loading.gif">
	position:2,//位置：角+排列 1：左上，2：右上，3：右下，4：左下,排列：1 2 从左到右，从上到下。
	mousez:10,//提示框和鼠标之间的距离
	mouse:0//鼠标跟随。1：左上，2：右上，3：左下，4：右下
}
